# 📝 Django Blog Application

## 🚀 Overview
This is a full-stack Blog Web Application developed using Django as part of my backend development journey.  
The project demonstrates how a real-world web application works by integrating frontend, backend, and database systems.

---

## ✨ Key Features

- 🔐 User Authentication (Register, Login, Logout)
- 📝 Create, Update, Delete Blog Posts (CRUD)
- 📊 User Dashboard for managing posts
- 🔑 Secure Password Reset via Email (Token-based authentication)
- ⚙️ Form Validation and Error Handling
- 🌐 Dynamic content rendering using Django Templates

---

## 📄 Application Pages

- Home (Index)
- Blog Detail Page
- About Page
- Contact Page
- User Dashboard
- Create / Update / Delete Post
- Register / Login / Logout
- Forgot Password / Reset Password
- Admin Panel

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL

---

## 🧠 Learning Outcomes

Through this project, I gained practical experience in:

- Understanding Django architecture and project structure
- Connecting frontend, backend, and database
- Implementing authentication and user management
- Handling forms and validations
- Building a secure password reset system using tokens
- Debugging real-world backend issues
